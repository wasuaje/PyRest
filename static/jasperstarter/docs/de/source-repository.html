Übersicht

Dieses Projekt verwendet Mercurial um seinen Quellcode zu verwalten. Informationen über Mercurial finden Sie unter: http://hgbook.red-bean.com/read/.

Web-Zugang

Es folgt ein Verweis auf das Web-Portal des Quellcode-Archivs dieses Projektes.

https://bitbucket.org/cenote/jasperstarter/src

Anonymer Zugang für jedermann

Der Quellcode dieses Projektes kann anonym mit Mercurial über folgenden Befehl ausgecheckt werden (Siehe http://www.selenic.com/mercurial/hg.1.html#clone).

$ hg clone https://bitbucket.org/cenote/jasperstarter/

Gesicherter Zugang für Entwickler

Nur Projekt Entwickler können mit dieser Methode auf den Mercurial Baum zugreifen. (Siehe http://www.selenic.com/mercurial/hg.1.html#clone).

$ hg clone ssh://hg@bitbucket.org/cenote/jasperstarter

Zugriff durch eine Firewall

Bitte konsultieren Sie die Dokumentation des verwendeten Versionskontrollsystems für weitere Informationen über den Zugriff durch eine Firewall.
